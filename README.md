
# News app API

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Dependencies

- Node <= v16.14.1
- MongoDB credentials
- JWT_KEY


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

```
PORT
MONGODB_CREDENTIALS
JWT_KEY
```
